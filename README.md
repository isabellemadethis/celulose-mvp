# Celulose MVP

App de produção de conteúdo da [Celulose Digital](https://celulose.digital). Multi-cliente, single-page HTML + Firebase Realtime Database.

## Acesso

O acesso é via link com PIN. Se você é cliente da Celulose Digital, peça o seu link à equipe.

## Stack

HTML + Vanilla JS + Firebase RTDB (compat SDK via CDN). Sem build step.
